## Iniciando Projeto

Primeiro, rode o servidor de desenvolvimento:
```bash
npm run dev
# ou
yarn dev
# ou
pnpm dev
# ou
bun dev
```
Abra [http://localhost:3000](http://localhost:3000) com seu navegador para ver o resultado.

## Algum erro?

*Erro no import do @heroui/react*
```bash
npm install @heroui/react
```
